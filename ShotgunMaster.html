Shotgun Master

Unity Engine - Solo Project

Overview

Shotgun Master is a 2D Platformer Game with a focus on game feel and juice.
The goal of the game is to get to the end of the level and defeat all enemies within.
Made as part of my second year at the University of Staffordshire, this project was created within the Unity Engine, using C# to create all mechanics.

KEY DESIGN FEATURES

Forgiving Controls

To create the feeling of smooth movement and a polished experience, many forgivessness mechanics have been implemented such as:
Coyote time, Jump Buffering, Variable Jump Height.

Powerful Movement Feel

The player is intentionally designed to feel heavy,

Exaggerated Visual Effects

If the winning player of a round achieves a score higher than any score in the current leaderboard, the game prompts the player for a name input and then stores that players name into the song leaderboard in the relevant position, accompanied with their score.
Each song has it's own leaderboard.

Platforming Mechanics

Each directional input corresponds to a dance animation for the players character, creating the effect that the character is dancing in time with the player inputs.
This creates a very fun and playful vibe for the game, as the character will still continue to dance even if the player is missing the notes.
