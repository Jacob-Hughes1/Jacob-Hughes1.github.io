Focus Tool

Standalone Customisable Focus Tool

Overview

This is a standalone tool designed to help users with time management for any tasks.
This tool features customisable timers accompanied by task and note management to help the user manage their time and tasks.


KEY FEATURES

Custom Tasks

The user can add custom tasks with a title, description and priority.
To ensure the user remains focussed, only a single task can ever be set as the current task, the user is intended to finish this task before selecting a new current task.

Pomodoro Technique Timer

The default state for the timer uses the Pomodoro technique as it's base, however this time can be customised if the user wishes.
The timer is designed to be used to allow the user to focus for a set amount of time, while taking regular breaks.

Custom Notes

Notes can be made by the user, if there is anything they would like to remember or keep track of.
Notes can be individually popped out into their own seperate windows and pinned on top of all other windows if required.

Rest Functionality

Short and Long Rests can be customised in their length and activated whenever the user wishes.
When the Rest timer ends, the main timer continues automatically. There is a pause function for any of the timers which can be used at any time.
